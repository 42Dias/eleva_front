### Starting...
#### Make a clone
```
git clone git@github.com:42Dias/eleva_front.git
```
### Install the dependencies
```
yarn
# or
npm install
```
### Usage
```
yarn dev
# or
npm run dev
```
Open http://localhost:3000 to view it in the browser.

* [Styled Components](https://styled-components.com)
